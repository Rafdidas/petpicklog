# 반려동물 생활 정보 플랫폼 화면 설계서 v0.5

## 1. 문서 목적

이 문서는 `반려동물 생활 정보 플랫폼 설계서 v0.5`를 기반으로 실제 화면 디자인을 진행하기 위한 화면 설계 문서이다.

기능 중심 설계가 아니라 디자이너 또는 프론트엔드 개발자가 바로 화면 구조를 잡을 수 있도록 아래 내용을 정리한다.

```txt
페이지별 목적
화면 레이아웃
섹션 구성
카드 UI 구성
주요 문구
상태별 화면
반응형 방향
디자인 톤
```

---

## 2. 전체 디자인 방향

## 2.1 서비스 인상

이 서비스는 단순 쇼핑몰이나 가격비교 사이트처럼 보이면 안 된다.

전체 인상은 아래에 가깝게 잡는다.

```txt
정보는 신뢰감 있게
화면은 가볍고 따뜻하게
행동은 쉽게 선택할 수 있게
```

사용자가 느껴야 하는 이미지는 다음과 같다.

```txt
반려동물과 함께 사는 사람이 자주 확인하는 생활 정보 앱
공공데이터 기반이라 출처가 명확한 서비스
가격 확인도 가능하지만 쇼핑몰처럼 과하게 판매 중심은 아닌 서비스
```

---

```txt
신뢰감
가벼운 탐색

지역 기반

```

```txt
쇼핑몰처럼 상품만 과하게 나열하는 화면
공공데이터 포털처럼 딱딱한 표 중심 화면
지도 앱처럼 복잡한 필터가 먼저 보이는 화면
커뮤니티 앱처럼 글/피드가 중심인 화면
```

## 2.4 추천 UI 톤

```txt
배경: 아주 연한 웜톤 또는 그레이 톤
카드: 흰색 또는 약한 배경색

```

# 4. 홈 화면 `/`

## 4.1 화면 목적

홈은 사용자가 서비스의 정체성을 바로 이해하고, 아래 두 가지 핵심 기능으로 진입하는 시작 화면이다.

```txt
1. 반려생활 장소 찾기
2. 반려동물 용품 가격 확인
```

---

## 4.2 화면 구성 요약

```txt
1. Hero / 통합 검색
2. 빠른 기능 카드
3. 반려생활 장소 찾기
4. 많이 찾는 용품
5. 내 관심상품 가격 변화
6. 최근 확인한 상품
7. 반려생활 가이드
8. 데이터 출처 안내
```

---

## 4.3 Hero / 통합 검색

### Desktop 레이아웃

```txt
┌─────────────────────────────────────────────────────────────┐
│ [Badge] 공공데이터 + 쇼핑 API 기반 반려생활 정보              │
│                                                             │
│ 반려동물과 함께하는 생활 정보를                              │
│ 한곳에서 확인하세요.                                         │
│                                                             │
│ 동반 여행지, 동물병원, 동물약국, 미용·위탁관리 업소 정보부터    │
│ 자주 사는 용품 가격까지 확인할 수 있어요.                     │
│                                                             │
│ [ 지역명, 장소명, 강아지 사료, 고양이 모래 검색        ][검색] │
│                                                             │
│ [장소 찾기] [가격 확인하기]                                  │
│                                                             │
│ 추천: 동물병원 · 동물약국 · 강아지 사료 · 고양이 모래          │
└─────────────────────────────────────────────────────────────┘
```

### Mobile 레이아웃

```txt
[Badge]
반려동물과 함께하는
생활 정보를 한곳에서 확인하세요.

동반 여행지, 동물병원, 동물약국, 용품 가격까지 확인할 수 있어요.

[검색창]
[장소 찾기]
[가격 확인하기]

추천 검색어 chip
```

### 디자인 포인트

```txt
Hero 영역은 너무 쇼핑몰처럼 상품 이미지를 크게 넣지 않는다.
반려동물 실루엣, 산책, 여행, 병원, 쇼핑을 추상화한 일러스트가 어울린다.
검색창을 크게 배치해서 정보 탐색 서비스라는 인상을 준다.
```

### 검색 동작

검색어가 장소 성격이면 `/places?keyword=검색어`로 이동한다.
상품 성격이면 `/products?query=검색어`로 이동한다.

초기에는 자동 판별이 어려우므로 버튼을 분리해도 된다.

```txt
검색창 + [장소에서 검색] [상품에서 검색]
```

MVP에서는 이 방식이 더 안전하다.

## 4.5 반려생활 장소 찾기 섹션

### 목적

### 레이아웃

```txt
[섹션 타이틀] 반려생활 장소 찾기
[서브] 공공데이터 기반으로 동반 여행지, 동물병원, 동물약국, 미용·위탁관리 업소를 확인해보세요.

[동반여행] [동물병원] [동물약국] [동물미용] [위탁관리]

┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐
│ 동반여행          │ │ 동물병원          │ │ 동물약국          │
│ 반려동물과 갈 수... │ │ 우리 동네 병원...  │ │ 약국 정보를 확인... │
│ [찾아보기]        │ │ [찾아보기]        │ │ [찾아보기]        │
└──────────────────┘ └──────────────────┘ └──────────────────┘
```

### 카테고리 카드 정보

```txt
카테고리명
짧은 설명
대표 아이콘
데이터 출처 배지
바로가기
```

### 카피 예시

```txt
동반여행
반려동물과 함께 갈 수 있는 여행지와 시설 정보를 확인해보세요.

동물병원
지역별 동물병원 정보를 확인하고 지도에서 위치를 찾아보세요.

동물약국
반려동물 의약품을 취급하는 약국 정보를 확인해보세요.

동물미용
가까운 동물미용 업소 정보를 찾아보세요.

위탁관리
호텔링, 돌봄 등 위탁관리 업소 정보를 확인해보세요.
```

## 4.6 많이 찾는 용품 섹션

```txt

[사료] [간식] [배변패드] [고양이 모래] [샴푸·위생] [영양제] [장난감] [이동장]
```

```txt
상품 카테고리는 칩 또는 작은 카드 형태가 적합하다.
이미지보다 텍스트와 아이콘 중심이 좋다.
```

# 5. 장소 찾기 화면 `/places`

공공데이터 기반 반려생활 장소·업소 정보를 검색하고 필터링하는 핵심 화면이다.

기존 `/hospitals` 화면을 확장한 형태로, MVP의 가장 중요한 신규 화면이다.

---

## 5.2 Desktop 레이아웃

```txt
[페이지 헤더]
반려생활 장소 찾기
공공데이터 기반으로 동반 여행지, 동물병원, 동물약국, 미용·위탁관리 업소 정보를 확인해보세요.

[검색 영역]
┌─────────────────────────────────────────────────────────────┐
│ [지역명, 장소명, 주소로 검색                         ][검색] │

└─────────────────────────────────────────────────────────────┘

[카테고리 탭]
전체 | 동반여행 | 동물병원 | 동물약국 | 동물미용 | 위탁관리

[결과 요약]
총 128개의 장소를 찾았어요.    [최신 동기화: 2026.06.10]

[본문]
```

# 15. 최종 디자인 판단

이번 서비스의 핵심은 `장소 찾기`와 `가격 확인`을 같은 플랫폼 안에서 자연스럽게 연결하는 것이다.

따라서 홈에서는 두 기능을 동일한 비중으로 보여주되, 신규 차별점인 공공데이터 기반 장소 정보를 조금 더 앞에 배치한다.

최종 화면 방향은 다음과 같다.

```txt
홈: 서비스 정체성 전달 + 주요 기능 진입
장소 찾기: 공공데이터 기반 검색과 카드형 정보
가격 확인: 네이버 쇼핑 기준 가격 확인과 관심상품 저장
관심상품: 저장 당시 가격과 현재 확인 가격 비교
가이드: 사용자의 판단을 돕는 정적 콘텐츠
```

이 구조로 가면 단순 가격비교 서비스가 아니라, 반려동물과 함께 사는 사람이 자주 확인하는 생활 정보 서비스처럼 보일 수 있다.
