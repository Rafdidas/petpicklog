# 펫픽 Korean Pick 로고 디자인

날짜: 2026-07-22
상태: 사용자 방향 승인

## 1. 목표

리뉴얼된 펫픽 UI에 맞는 한글 중심 로고 시스템을 만든다. 로고는 반려동물 쇼핑몰보다 **매일 가격을 기록하고 더 나은 선택을 돕는 서비스**로 먼저 인식되어야 한다.

핵심 인상은 영리함, 단정함, 신뢰감, 절제된 친근함이다.

## 2. 현재 UI 맥락

로고는 현재 디자인 토큰과 헤더 구조를 기준으로 한다.

- 캔버스: 밝은 블루그레이 `#f3f6fa`
- 기본 표면: 흰색 `#ffffff`
- 기본 잉크: 딥 네이비 `#17222b`
- 브랜드 포인트: 라임 `#bfdf74` 및 로고 배경 `#c3e17e`
- 형태 언어: 둥근 카드와 컨트롤, 가벼운 그림자, 친근한 서비스 UI
- 기본 폰트: Pretendard Variable
- 헤더의 기존 로고 영역: 34px 정사각형 마크 + `펫픽` + 보조 설명

## 3. 선택한 방향: Korean Pick

한글 `펫픽`을 중심으로 한 커스텀 워드마크를 사용한다. Pretendard와 자연스럽게 어울리는 골격을 유지하면서 일부 획만 조정해 독자성을 만든다.

- 두 글자의 `ㅍ` 가로획은 가격 데이터의 기준선처럼 안정적으로 정렬한다.
- 마지막 `픽`의 `ㄱ`에는 작은 틱 형태를 적용해 기록과 선택의 의미를 암시한다.
- 의미 장치는 작은 크기에서도 글자 판독을 해치지 않는 범위로 제한한다.
- 기본 워드마크는 딥 네이비로 표현하고 라임은 한 곳의 작은 포인트에만 사용한다.

## 4. 로고 구성

### 기본 가로형

- 커스텀 한글 워드마크 `펫픽`
- 기본 색상은 `#17222b`
- 웹 헤더에서 보조 문구 `반려용품 가격추적`과 나란히 사용할 수 있어야 한다.
- 작은 헤더에서도 획이 뭉치거나 라임 포인트가 사라지지 않아야 한다.

### 축약 심볼

- 커스텀 `ㅍ`을 독립 심볼로 사용한다.
- 기본형은 `#c3e17e` 배경 위 `#17222b` 심볼이다.
- 현재 34px 정사각형 헤더 마크를 직접 교체할 수 있어야 한다.
- 파비콘과 앱 아이콘에서는 보조 텍스트 없이 식별 가능해야 한다.

### 단색형

- 검정 또는 흰색 한 색만으로도 형태와 판독성이 유지되어야 한다.
- 인쇄, 작은 워터마크, 색상 제약이 있는 환경에서 사용한다.

## 5. 시각 원칙

- 귀엽기보다 명료하고 영리하게 보인다.
- 반려동물 카테고리는 서비스명과 문맥으로 전달하고 흔한 펫숍 상징에 의존하지 않는다.
- 발바닥, 하트, 동물 얼굴, 뼈다귀, 가격표 실루엣은 사용하지 않는다.
- 복잡한 그래프, 가는 선, 그라데이션, 입체 효과는 사용하지 않는다.
- 라임은 브랜드 식별 포인트로 제한하고 넓은 면적의 워드마크 전체에 적용하지 않는다.

## 6. 산출물

- 기본 가로형 로고 SVG
- 축약 심볼 SVG
- 단색 가로형 SVG
- 파비콘용 SVG 또는 PNG
- 실제 헤더에 적용한 반응형 상태

SVG를 원본으로 삼아 크기와 색상을 안정적으로 제어한다. 생성형 이미지 결과는 형태 탐색 참고용으로만 사용할 수 있으며, 한글 글자와 최종 획은 코드 기반 벡터로 정리한다.

## 7. 적용 및 검증

- `Header`의 기존 영문 `P` 마크를 축약 심볼로 교체한다.
- 데스크톱에서 워드마크와 보조 문구의 기준선 및 간격을 확인한다.
- 모바일에서 보조 문구가 사라져도 로고가 자연스럽게 보여야 한다.
- 밝은 헤더, 딥 네이비 배경, 단색 환경에서 대비를 확인한다.
- 16px 파비콘, 34px 헤더 심볼, 일반 데스크톱 워드마크 크기에서 판독성을 확인한다.

## 8. 범위 제외

- 캐릭터 또는 마스코트 개발
- 브랜드 전체 카피와 네이밍 변경
- UI 색상 토큰의 전면 개편
- 인쇄물과 패키지 디자인
